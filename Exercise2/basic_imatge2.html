Question:

What is the difference between the real size of the image and these attributes?

Answer:

The width and height attributes control how big the image appears on the webpage, but they don't change the actual size of the image file. So, if you set the width to 300 pixels and height to 200 pixels, the image will display at that size on the webpage, but the original image file might be bigger or smaller.
